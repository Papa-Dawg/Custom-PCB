## Overview
This is a custom PCB designed in **KiCad 9.0**. It is an Arduino-compatible controller featuring:
* **7-Segment Display:** Integrated layout for numerical feedback.
* **Breakout Ports:** Easy access to Port B, C, and D for prototyping.
* **Onboard Controls:** Includes a Reset button and a programmable button.
* **Power Management:** Integrated On/Off toggle and LED indicator.

## Project Structure
* `/Arduino1.kicad_sch`: Schematic design.
* `/Arduino1.kicad_pcb`: Board layout and routing.
* `3-D Model.png`: Visual render of the completed board.
